## 使用说明

本知识库服务于 AddictionQuitr App 的内容体系建设，可用于：

1. **App 内容素材** — 提炼书籍和论文中的核心方法论，融入 App 的每日任务和指导内容
2. **科学依据支撑** — 引用学术论文中的神经科学研究成果，增强 App 的专业可信度
3. **RAG 知识检索** — 作为 AI 助手的知识库，为用户提供个性化的智能问答
4. **多语言覆盖** — 中英文资源兼备，支持国际化内容运营
